깃헙 편의상 .py로 저장했으며 구동을 위해서는 .Entry 확장자 변환과 엔트리파이썬 프로그램이 필요함.<br />
이 소스코드들은 이동현이 제작한 것으로 복사하여 응용하는 것은 가능하나 그대로 과제 제출하는 사람이 있을 경우 사람 취급 안 함.<br />

1. holzzak.py
- 숫자를 입력받아 홀수/짝수 출력(반복)
- 0을 입력하면 프로그램 종료

2. 1~b=nani.py
- 숫자(1~100) 입력하여 1부터 입력한 숫자까지의 합 출력하기
- 0을 입력하면 종료
- 음수 또는 100 초과 값을 입력하면 "잘못된 입력" 출력하기.

3. a~bplus.py
- a값과 b값을 받아 a값부터 b값까지 순차적으로 더하여 출력함.
